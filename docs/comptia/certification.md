## Certification

- Release date April 20, 2022.
- Both exams are required for A+ Certification
- Pass exams before they retire (3.5 years)
- No order and no time restricitons between exams
- Information available at CompTIA website

| Core | Name |
| ----------- | ----------- |
| 1 | [220-1101]() |
| 2 | [220-1102]() |

### CompTIA A+ (Core 1)
- Mobile Devices
- Networking
- Hardware
- Virtualization and Cloud Computing
- Hardware & Network Troubleshooting

### CompTIA A+ (Core 2)
- Operating Systems
- Security
- Software Troubleshooting
- Operational Procedures

### Exam
- 90 mins
- 90 questions max
- Passing score core 1/2 675/700 (on a scale of 100 - 900)
- Multiple choice, straight forward single multiple answers
- Performance-based
    - Complete a task
    - Command line, matching, sorting, drag-and-drop etc.
- CompTIA Exam objectives available on website (study material for what will be on the test)
