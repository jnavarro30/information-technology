Frequency bands are radio waves that carry data, this is how wireless data is transfered between devices.

| Frequency Band | Description |
| ----------- | ----------- |
| 2.4 GHz | Most common with other devices (fridge, microwave, smart devices), potentialy more interference, slower speed longer range |
| 5 GHz | Less common, minimal interference, 25 non overlapping channels, faster speed shorter range. Difficulty penetrating solid objects like walls |

Wireless channels - fine tune and alter frequency

WiFi
- connect devices to internet
- Range 30 - 91 meters (100 - 300 feet)

Bluetooth
- connect devices to each other and exchange data
- slower transfer rate and shorter range vs wifi
- less power, longer battery life
- less vulnerable to interference
- uses FHSS (Frequency Hopping Spread Spectrum) less likely to get interference from other 2.4 GHz devices unlike wiFi
- simpler to use/ often no password required
- headsets, speakers, keyboards/mice
- uses the 2.4 GHz range
- Unlicensed ISM (Industrial, Scientific and Medical) band
- Same as 802.11
Short-range
- mos consumer devices operate to about 10 meters (30 feet)

RFID (Radio-frequency identification)
- access badges
- inventory/assembly line tracking
- pet/animal identification
- anything that needs to be tracked
Radar Technology
- radio enery transmitted to the tag
- RF powers the tag (no battery, not powered alone), ID is transmitted back
- Bidirectional communication
- Some tag formats can be active/powered (some do have batteries) and can be tracked a longer distance away
NFC (Near Field Communication)
- two-way communication
- builds 0n RFID, which is mostly one-way
payment systmes
- major credit cards, online wallets
boostrap for other wireless
- NFC helps with Bluetooth pairing 
Access token, identity 'card'
- short range with encryption support
